# uniapp脚手架

## 安装依赖
```
yarn install
```

### 本地运行
```
yarn serve
```

### 打包
```
yarn build
```

### master分支为HBuilderX版本，默认分支为cli版本
### 本脚手架主要用于微信小程序开发
